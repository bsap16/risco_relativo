# 📊 Análise de Risco de Inadimplência — Super Caja  
## *Automatização de Aprovação de Empréstimos com Machine Learning*

---

# 🇧🇷 Português

## 📌 Visão Geral
Este projeto visa **avaliar o risco relativo de inadimplência** de clientes do banco Super Caja em operações de crédito, utilizando dados históricos e técnicas de machine learning para **automatizar decisões de aprovação de empréstimos**.

---

## 🎯 Objetivo
Desenvolver um modelo preditivo para:

1. Identificar padrões de comportamento associados à inadimplência.  
2. Classificar clientes em categorias de risco (baixo, médio e alto).  
3. Otimizar o processo de análise de crédito com base em dados.  

---

## 🛠️ Ferramentas Utilizadas

### Processamento de Dados
- SQL (BigQuery)

### Análise e Machine Learning
- Python (Google Colab)

### Visualização
- Looker Studio

### Bibliotecas
```python
Pandas
NumPy
SciPy
Scikit-learn
Matplotlib
Seaborn
```

---

## 🔍 Principais Etapas

### 1. Preparação dos Dados
- Limpeza de outliers e tratamento de valores faltantes via SQL  
- Engenharia de features, como:
  - Score de renda  
  - Histórico de pagamentos  
  - Variáveis derivadas para risco de crédito

---

### 2. Análise Exploratória
- Distribuições de:
  - Idade  
  - Renda  
  - Motivo do empréstimo  

- Correlação entre variáveis e inadimplência

---

### 3. Modelagem Preditiva
Treinamento e avaliação com:

- Regressão Logística  
- Balanceamento com SMOTE  
- Matriz de Confusão  

### Métricas de Validação
- Acurácia  
- Precision  
- Recall  
- F1-score  
- AUC-ROC

# 🇺🇸 English
# 📊 Loan Default Risk Analysis — Super Caja  
## *Automating Loan Approval Decisions with Machine Learning*


## 📌 Overview
This project aims to **assess the relative risk of loan default** among Super Caja customers using historical data and machine learning techniques to **automate loan approval decisions**.

---

## 🎯 Objective
Develop a predictive model to:

1. Identify behavioral patterns associated with default risk.  
2. Classify customers into risk categories (low, medium, high).  
3. Optimize the credit analysis process through data-driven decision-making.  

---

## 🛠️ Tools Used

### Data Processing
- SQL (BigQuery)

### Analytics and Machine Learning
- Python (Google Colab)

### Visualization
- Looker Studio

### Libraries
```python
Pandas
NumPy
SciPy
Scikit-learn
Matplotlib
Seaborn
```

---

## 🔍 Key Project Stages

### 1. Data Preparation
- Outlier cleaning and missing value treatment using SQL  
- Feature engineering, including:
  - Income score  
  - Payment history  
  - Derived credit risk variables

---

### 2. Exploratory Data Analysis
Visualization of distributions for:

- Age  
- Income  
- Loan purpose  

Analysis of correlations between variables and default behavior.

---

### 3. Predictive Modeling
Model training and evaluation using:

- Logistic Regression  
- SMOTE balancing  
- Confusion Matrix

### Validation Metrics
- Accuracy  
- Precision  
- Recall  
- F1-score  
- AUC-ROC
